Godot Engine

Free & open source 2D and 3D game engine

The powerful, feature-rich game engine that powers UF-OS's development tools and capabilities. With built-in scripting, visual editor, and cross-platform deployment, Godot provides the foundation for modern game development workflows.

Universal Blue

Cloud-native operating system images

Built on Fedora Atomic and container technology, UBlue provides reliable, image-based operating systems with atomic updates and rollbacks. This foundation ensures UF-OS stays secure, stable, and always up-to-date.

How to Set Up Fedora Atomic Desktop For UF-OS/StatOS

  1. Download an official Fedora Atomic Desktop ISO:
    Flash the ISO to a USB drive and install Fedora as usual.
    We recommend installing with Kinoite, but any Fedora Atomic Desktop will work.
  2. Rebase to the unsigned variant of the Universal Blue base image:
    Open a terminal and run:
    rpm-ostree rebase ostree-unverified-registry:ghcr.io/uf-os/statos:latest
    Replace uf-os/statos with your chosen image if different.
    Then reboot:
    systemctl reboot
  3. Rebase to the signed variant of the image:
    After rebooting, run:
    rpm-ostree rebase ostree-image-signed:docker://ghcr.io/uf-os/statos:latest
    Again, replace uf-os/statos with your chosen image if needed.
    Then reboot again:
    systemctl reboot
Source: Universal Blue: How to Install Base Images
